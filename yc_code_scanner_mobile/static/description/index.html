Mobile Barcode/Qrcode Scanner widgets

This feature is available only in secure contexts (HTTPS)

Description

Enhance your Odoo experience with a powerful Mobile Barcode and QR Code Scanner, designed for seamless document and product management across various modules. This all-in-one solution integrates barcode scanning capabilities into Odoo’s sales, purchase, inventory, accounting, manufacturing, and event management workflows.

Key Features:
Scan Using Mobile Camera & Webcam: Effortlessly scan barcodes and QR codes using a smartphone camera or webcam for real-time data entry.
Global Search by Barcode & QR Code: Quickly retrieve any document across Odoo by scanning its barcode or QR code.
Search & Select in Many2One & Many2Many Fields: – Instantly search and populate Many2One and Many2Many fields using barcode and QR code scanning.
Character Field Scanner Widget: Enable barcode and QR code scanning for character fields, streamlining data input.
Boolean Field Scanner Widget: Enables barcode and QR code scanning for Boolean fields, allowing users to verify a scanned code against a specified code field. It also provides the flexibility to trigger actions based on successful or failed verification through configurable widget options.
Flexible & Easy-to-Use: Works with built-in mobile camera, eliminating the need for additional scanning hardware.

This module provides a fully integrated mobile scanning solution for Odoo, enabling businesses to streamline operations and enhance productivity with barcode and QR code automation.

Installation & Configuration
Installation & Widget Usage

After installing the module, use the following widgets for specific field types:
1. Char Field: `yc_char_barcode_scanner`
2. Many2One Field: `yc_many2one_barcode_scanner`
3. Many2Many Field: `yc_many2many_tags_code_scanner`
4. Boolean Field: `yc_boolean_scanner`
Boolean Field Widget Options:
- `accepted_barcode_field` *(Required)* – Specify the barcode field to match the scanned barcode for validation.
- `action_perform_on_successful` – Define the action or method to be triggered when the barcode is valid.
- `action_perform_on_failure` – Define the action or method to be triggered when the barcode is invalid.
After applying the widget to a specific field, the view will appear as shown in the following screenshot.
Click the Scan button to initiate the code scanning process.
Upon successful scanning, the popup will close automatically, and the field's value will be populated based on the field type. Additionally, a success message will be displayed on the right side of the screen.
Effortless Search with Barcode & QR Code Scanner

Seamlessly search any document globally using the Barcode and QR Code scanner in the search view. This feature enhances warehouse operations, sales and purchase order processes, accounting, product lot/serial number tracking, inventory adjustments, manufacturing workflows, and event management — all available out of the box. No additional ERP integration is required for barcode and QR code scanning.
